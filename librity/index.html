Librity

— Visual Identity

— Librity is a project dedicated to the digitization of the book market. It will be a website/app where e-books and audiobooks will be sold.

Preparation

— 01

— The target group for this project is all ages. Since it is such a "library" in an online environment, it adds features of formality, technology/digitization and modern look.

— The dark blue color in the background is to represent digitalization/technology, formal and modern look.

Design of visual identity

— 02

— The logo consists of typography and vector elements.

— The first one is the name of the project (Librity). I chose a clean font for the name because it looks formal and modern.

— The second part of the logo is the vector elements. On the left side, the lines represent sound waves, therefore defining the audiobooks. On the right side, the lines lean on each other, therefore defining the books leaning on each other.

Final Look

— 03

Conclusion

— 04

— The logo as a whole should inform about the name of the project and define that it is the sale of e-books and audiobooks in digital form.
